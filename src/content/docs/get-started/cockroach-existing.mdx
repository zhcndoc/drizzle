# 在现有项目中使用 Drizzle 和 CockroachDB 入门

<Callout type='error'>
本页面解释的是在 drizzle 版本 `1.0.0-beta.2` 及更高版本中可用的概念。
</Callout>

<br/>

<Prerequisites>
  - **dotenv** - 用于管理环境变量的包 - [查看这里](https://www.npmjs.com/package/dotenv)
  - **tsx** - 用于运行 TypeScript 文件的包 - [查看这里](https://tsx.is/)
  - **node-postgres** - 用于查询你的 PostgreSQL 数据库的包 - [查看这里](https://node-postgres.com/)
</Prerequisites>

<FileStructure/>

#### 第 1 步 - 安装 **node-postgres** 包
<Npm>
  drizzle-orm@beta pg dotenv
  -D drizzle-kit@beta tsx @types/pg
</Npm>


#### 第 2 步 - 设置连接变量

<SetupEnv env_variable='DATABASE_URL' />

<Callout title='提示'>
如果你还没有 PostgreSQL 数据库，且想创建一个用于测试，可以使用我们的 Docker 中设置 PostgreSQL 的指南。

Docker 中的 PostgreSQL 指南见 [这里](/docs/guides/postgresql-local-setup)。按照指南完成设置并生成数据库 URL（指南中有详细说明），完成后返回继续下一步。
</Callout>

#### 第 3 步 - 设置 Drizzle 配置文件

<SetupConfig dialect='cockroach' env_variable='DATABASE_URL'/>

<IntrospectCockroach/>

<TransferCode/>

<ConnectCockroach/>

#### 第 7 步 - 查询数据库
