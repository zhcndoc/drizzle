# 使用 Drizzle 和 MSSQL 入门

<Callout type='error'>
本页面讲解适用于 drizzle 版本 `1.0.0-beta.2` 及以上的概念。
</Callout>

<br/>

<Prerequisites>
  - **dotenv** - 管理环境变量的包 - [查看这里](https://www.npmjs.com/package/dotenv)
  - **tsx** - 运行 TypeScript 文件的包 - [查看这里](https://tsx.is/)
  - **node-mssql** - 查询 MSSQL 数据库的包 - [查看这里](https://github.com/tediousjs/node-mssql)
</Prerequisites>

Drizzle 原生支持使用 `node-mssql` 驱动连接 PostgreSQL。
