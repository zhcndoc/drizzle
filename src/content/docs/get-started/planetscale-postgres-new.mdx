# 开始使用 Drizzle 和 PlanetScale Postgres

<Prerequisites>
  - **dotenv** - 管理环境变量的包 - [查看这里](https://www.npmjs.com/package/dotenv)
  - **tsx** - 运行 TypeScript 文件的包 - [查看这里](https://tsx.is/)
  - **PlanetScale Postgres** - PostgreSQL 数据库平台 - [查看这里](https://planetscale.com/docs/postgres)
  - **node-postgres** - 用于查询 PostgreSQL 数据库的包 - [查看这里](https://node-postgres.com/)
</Prerequisites>

<Callout title='PlanetScale 同时支持 MySQL' type='info'>
在找 MySQL 吗？查看我们的[PlanetScale MySQL 指南](/docs/get-started/planetscale-new)
</Callout>

PlanetScale 同时提供 MySQL（Vitess）和 PostgreSQL 数据库。本指南涵盖如何使用标准的 `node-postgres` 驱动连接 PlanetScale Postgres。

关于如何创建 PlanetScale Postgres 数据库和获取凭证的详细说明，请参阅[PlanetScale Postgres 文档](https://planetscale.com/docs/postgres/tutorials/planetscale-postgres-drizzle)。

<FileStructure/>

#### 步骤 1 - 安装 **node-postgres** 包
<InstallPackages lib='pg' devlib=' @types/pg'/>

#### 步骤 2 - 配置连接变量

在项目根目录下创建 `.env` 文件，并添加数据库连接变量：

```plaintext copy
DATABASE_URL=postgresql://{username}:{password}@{host}:{port}/postgres?sslmode=verify-full
```

<Callout title='提示'>
你可以通过访问 PlanetScale 仪表板，进入你的数据库，点击 **“Connect”** 并创建 **“Default role”** 来获取连接凭证。详细步骤请参阅[PlanetScale 连接指南](https://planetscale.com/docs/postgres/tutorials/planetscale-postgres-drizzle#create-credentials-and-connect)。
</Callout>
