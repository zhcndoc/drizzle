# 使用 Drizzle 和 CockroachDB 入门

<Callout type='error'>
此页面介绍的概念适用于 drizzle 版本 `1.0.0-beta.2` 及更高版本。
</Callout>

<br/>

<Prerequisites>
  - **dotenv** - 管理环境变量的包 - [查看这里](https://www.npmjs.com/package/dotenv)
  - **tsx** - 运行 TypeScript 文件的包 - [查看这里](https://tsx.is/)
  - **node-postgres** - 访问 PostgreSQL 数据库的包 - [查看这里](https://node-postgres.com/)
</Prerequisites>

Drizzle 原生支持使用 `node-postgres` 和 `postgres.js` 驱动连接 CockroachDB。

本入门示例将使用 `node-postgres`。如果你想了解更多连接 PostgreSQL 的方式，请查看我们的 [CockroachDB 连接](/docs/get-started-cockroach) 页面。
