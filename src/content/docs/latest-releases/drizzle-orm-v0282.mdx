## 社区贡献发布 🎉

### 内部功能与变更
- 为 d1 添加了一组测试
- 修复了内部文档中的问题

### 修复
- 解决了 MySQL 时间戳毫秒截断的问题
- 纠正了 sqlite 基础方言中 `.get()` 方法的类型 ([#565](https://github.com/drizzle-team/drizzle-orm/issues/565))
- 修正了导致查询执行两次的 sqlite-proxy 错误

### 新包 🎉

在 [drizzle-typebox](/docs/typebox) 包中添加了对 [Typebox](https://github.com/sinclairzx81/typebox) 的支持。

请查看文档页面以获取更多使用示例：/docs/typebox
