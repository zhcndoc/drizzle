## 新功能

- 为 Neon HTTP 驱动程序中的批处理 API 添加了原始查询支持 (`db.execute(...)`)

要开始使用 Neon 和 Drizzle，请参考 [文档](/docs/get-started-postgresql#neon)

## 修复

- 修复了 `@neondatabase/serverless` HTTP 驱动程序类型问题 ([#1945](https://github.com/drizzle-team/drizzle-orm/issues/1945), [neondatabase/serverless#66](https://github.com/neondatabase/serverless/issues/66))
- 修复了 sqlite-proxy 驱动程序 `.run()` 结果

要开始使用 SQLite 代理驱动程序和 Drizzle，请参考 [文档](/docs/get-started-sqlite#http-proxy)
