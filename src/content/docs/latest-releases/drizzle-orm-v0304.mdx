## 新功能

### 🎉 支持 xata-http 驱动

根据他们的 **[官方网站](https://xata.io)**，Xata 是一个专注于可靠性、可扩展性和开发者体验的 Postgres 数据平台。目前 Xata Postgres 服务处于 beta 阶段，请参阅 [Xata 文档](https://xata.io/docs/postgres) 了解如何在您的账户中启用它。

Drizzle ORM 原生支持 `drizzle-orm/xata` 包的 `xata` 驱动，以及访问 Xata Postgres 数据库的 **`postgres`** 或 **`pg`** 驱动。

以下示例使用 Xata 生成的客户端，您可以通过运行 [xata init](https://xata.io/docs/getting-started/installation) CLI 命令获得。

<Npm>
drizzle-orm @xata.io/client
-D drizzle-kit
</Npm>

```ts
import { drizzle } from 'drizzle-orm/xata-http';
import { getXataClient } from './xata'; // 生成的客户端

const xata = getXataClient();
const db = drizzle(xata);

const result = await db.select().from(...);
```

您还可以使用 `pg` 或 `postgres.js` 驱动连接到 Xata。

要开始使用 Xata 和 Drizzle，请参阅 [文档](/docs/get-started-postgresql#xata)。
