# Drizzle \<\> Turso

<Prerequisites>
- 数据库 [连接基础](/docs/connect-overview) 与 Drizzle
- Turso 数据库 - [网站](https://docs.turso.tech/introduction)
- LibSQL 驱动 - [网站](https://docs.turso.tech/sdk/ts/reference) & [GitHub](https://github.com/tursodatabase/libsql-client-ts)
</Prerequisites>

根据 **[官方网站](https://turso.tech/drizzle)**， 
Turso 是一个 **[libSQL](https://github.com/libsql/libsql)** 驱动的边缘 SQLite 数据库服务。
  
Drizzle ORM 原生支持 libSQL 驱动， 
我们拥抱 SQL 方言以及特定方言的驱动和语法，并且镜像最流行的 
类似 SQLite 的 `all`、`get`、`values` 和 `run` 查询方法语法。 

#### 第一步 - 安装包
<Npm>
drizzle-orm @libsql/client
-D drizzle-kit
</Npm>

#### 第二步 - 初始化驱动并执行查询
```typescript copy
import { drizzle } from 'drizzle-orm/connect';

const db = await drizzle('libsql', { connection: {
  url: process.env.DATABASE_URL, 
  authToken: process.env.DATABASE_AUTH_TOKEN 
}});

const result = await db.select().from(...)
```

如果需要同步连接，可以使用我们的附加连接 API， 
在其中指定驱动连接并将其传递给 Drizzle 实例。

```typescript copy
import { drizzle } from 'drizzle-orm/libsql';
import { createClient } from '@libsql/client';

const client = createClient({ 
  url: process.env.DATABASE_URL,
  authToken: process.env.DATABASE_AUTH_TOKEN
});

const db = drizzle(client);

const result = await db.select().from(users).all()
```
