## 新功能

### 🎉 支持 xata-http 驱动程序

根据官方网站[（https://xata.io）](https://xata.io)，Xata 是一个以可靠性、可伸缩性和开发者体验为重点的 Postgres 数据平台。Xata Postgres 服务目前处于 beta 版阶段，请参阅[Xata 文档](https://xata.io/docs/postgres)了解如何在您的账户中启用它。

Drizzle ORM 原生支持使用 `drizzle-orm/xata` 包的 `xata` 驱动程序以及用于访问 Xata Postgres 数据库的 **`postgres`** 或 **`pg`** 驱动程序。

以下示例使用由运行[xata init](https://xata.io/docs/getting-started/installation) CLI 命令生成的 Xata 生成的客户端。

<Npm>
drizzle-orm @xata.io/client
-D drizzle-kit
</Npm>

```ts
import { drizzle } from 'drizzle-orm/xata-http';
import { getXataClient } from './xata'; // 生成的客户端

const xata = getXataClient();
const db = drizzle(xata);

const result = await db.select().from(...);
```

您也可以使用 `pg` 或 `postgres.js` 驱动程序连接到 Xata

要开始使用 Xata 和 Drizzle，请参阅[文档](/docs/get-started-postgresql#xata)。
