## 新功能

### 🎉 OP-SQLite 驱动程序支持

使用示例

```ts copy {4,5,6,8}
import { open } from '@op-engineering/op-sqlite';
import { drizzle } from 'drizzle-orm/op-sqlite';

const opsqlite = open({
	name: 'myDB',
});

const db = drizzle(opsqlite);

await db.select().from(users);
```

有关更多用法和设置详细信息，请查看我们的[op-sqlite文档](http://orm.drizzle.team/docs/get-started-sqlite#op-sqlite)

- 修复了 Expo 驱动程序的迁移钩子
