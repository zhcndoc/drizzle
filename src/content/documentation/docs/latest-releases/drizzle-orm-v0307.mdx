## 修复

- 添加了对 `@vercel/postgres` 包的映射

在[此处](https://vercel.com/docs/storage/vercel-postgres)了解更多关于 Vercel Postgres 的信息。要开始使用 Drizzle 和 Vercel Postgres，请参阅[文档](/docs/get-started-postgresql#vercel-postgres)。

- 修复了 neon 驱动程序的间隔映射 - [#1542](https://github.com/drizzle-team/drizzle-orm/issues/1542)

在[此处](https://neon.tech/)了解更多关于 Neon 的信息。要开始使用 Drizzle 和 Neon，请参阅[文档](/docs/get-started-postgresql#neon)。
