# 表模式

<IsSupportedChipGroup chips={{ 'PostgreSQL': true, 'MySQL': true, 'SQLite': false }} />

Drizzle ORM 为 PostgreSQL 和 MySQL 方言提供了声明 SQL 模式的 API。

如果在 schema 中声明了一个实体，查询生成器将在查询中添加 schema 名称：<br/>
`select * from "schema"."users"`

<Callout type='warning' emoji='⚠️'>
  在 Postgres 中自定义 schema 中支持枚举目前仅在 `drizzle-orm` 中可用，对于 `drizzle-kit` 的支持即将推出。
</Callout>

<Tabs items={['PostgreSQL', 'MySQL', "SQLite"]}>
  <Tab>
    <Section>
    ```ts copy {3,5,7}
    import { serial, text, pgTable, pgSchema } from "drizzle-orm/pg-core";

    export const mySchema = pgSchema("my_schema");

    export const colors = mySchema.enum('colors', ['red', 'green', 'blue']);

    export const mySchemaUsers = mySchema.table('users', {
      id: serial('id').primaryKey(),
      name: text('name'),
      color: colors('color').default('red'),
    });

    
    ```
    ```sql
    CREATE SCHEMA "my_schema";

    CREATE TYPE "my_schema"."colors" AS ENUM ('red', 'green', 'blue');

    CREATE TABLE "my_schema"."users" (
      "id" serial PRIMARY KEY,
      "name" text,
      "color" "my_schema"."colors" DEFAULT 'red'
    );
    ```
    </Section>
  </Tab>
  <Tab>
    <Section>
    ```ts {3,5}
    import { int, text, mysqlTable, mysqlSchema } from "drizzle-orm/mysql-core";

    export const mySchema = mysqlSchema("my_schema")

    export const mySchemaUsers = mySchema.table("users", {
      id: int("id").primaryKey().autoincrement(),
      name: text("name"),
    });
    ```
    ```sql
    CREATE SCHEMA "my_schema";

    CREATE TABLE "my_schema"."users" (
      "id" serial PRIMARY KEY,
      "name" text
    );
    ```
    </Section>
  </Tab>
  <Tab>
  SQLite 不支持模式 😕
  </Tab>
</Tabs>
