## 新特性

- 在 Neon HTTP driver 的批处理 API 中添加了原始查询支持 (`db.execute(...)`)。

要开始使用 Neon 和 Drizzle，请参阅[文档](/docs/get-started-postgresql#neon)。

## 修复

- 修复了 `@neondatabase/serverless` HTTP driver 的类型问题 ([#1945](https://github.com/drizzle-team/drizzle-orm/issues/1945), [neondatabase/serverless#66](https://github.com/neondatabase/serverless/issues/66))。
- 修复了 sqlite-proxy driver 的 `.run()` 结果。

要开始使用 SQLite 代理驱动和 Drizzle，请参阅[文档](/docs/get-started-sqlite#http-proxy)。
