## 新特性

### 🎉 在所有 WHERE 表达式中添加了 `.if()` 函数

#### 选择所有视图数量大于 100 的帖子

```ts {5}
async function someFunction(views = 0) {
  await db
    .select()
    .from(posts)
    .where(gt(posts.views, views).if(views > 100));
}
```

## Bug 修复

- 修复了 AWS DataAPI 中对会话 `.all`、`.values`、`.execute` 函数的内部映射

在[文档](/docs/get-started-postgresql#aws-data-api)中阅读如何使用 AWS DataAPI 入门指南。
