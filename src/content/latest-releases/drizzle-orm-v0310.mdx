## 新功能

### 🎉 为所有 WHERE 表达式添加了 `.if()` 函数

#### 选择所有浏览量大于 100 的帖子

```ts {5}
async function someFunction(views = 0) {
  await db
    .select()
    .from(posts)
    .where(gt(posts.views, views).if(views > 100));
}
```

## 错误修复

- 修复了 AWS DataAPI 中会话 '.all'、'.values'、'.execute' 函数的内部映射问题

在[文档](/docs/get-started-postgresql#aws-data-api)中阅读使用 AWS DataAPI 快速入门指南。
