在 `src` 目录下创建一个 `index.ts` 文件并初始化连接：

```typescript copy
import { drizzle } from 'drizzle-orm/vercel-postgres';

const db = drizzle();
```

如果你需要提供你现有的驱动：

```typescript copy
import { sql } from '@vercel/postgres';
import { drizzle } from 'drizzle-orm/vercel-postgres';

const db = drizzle({ client: sql })
```
